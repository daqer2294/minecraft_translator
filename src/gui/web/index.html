Приложение не подписано — это нормально

      

        Это бесплатное приложение с открытым исходным кодом. Оно не куплено у
        Apple/Microsoft для «цифровой подписи», поэтому при первом запуске
        система может показать предупреждение. Это не вирус — просто система
        осторожничает. Ниже — что нажать.
      


      

        
🪟 Windows: «Система Windows защитила ваш компьютер»

        
    
          
  1. Нажмите «Подробнее» в окне предупреждения.
    2. 
          
  2. Внизу появится кнопка «Выполнить в любом случае» — нажмите её.
    3. 
          
  3. Приложение запустится. В следующий раз предупреждения не будет.
    4. 
        

        
[скриншот: окно SmartScreen → «Подробнее» → «Выполнить в любом случае»]

      


      

        
🍎 macOS: «Не удаётся проверить разработчика» / «Приложение повреждено»

        
Способ 1 — через настройки (проще):

        
    
          
  1. Попробуйте открыть приложение обычным двойным щелчком (появится предупреждение — закройте его).
    2. 
          
  2. Откройте Системные настройки → Конфиденциальность и безопасность.
    3. 
          
  3. Пролистайте вниз — там будет строка про «MinecraftTranslator» и кнопка
              «Всё равно открыть». Нажмите её и подтвердите.
    4. 
        

        
Способ 2 — если пишет «повреждено» (снять «карантин»):

        
    
          
  1. Откройте приложение Терминал.
    2. 
          
  2. Введите команду (замените путь на реальный) и нажмите Enter:
    3. 
        

        
xattr -cr /Applications/MinecraftTranslator.app

        
[скриншот: Конфиденциальность и безопасность → «Всё равно открыть»]
